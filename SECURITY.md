# 安全政策

## 报告漏洞

如果你发现了一个安全漏洞（比如代码注入、敏感信息泄露、依赖包漏洞等），**请不要公开提交 Issue**。

请通过以下方式联系我们：

- 邮箱：`143872820+bcht@users.noreply.github.com`(常看)或`bcht1145@gmail.com`

- GitHub Security Advisories 报告：< https://github.com/bcht/Elaina/security/advisories >

我们会尽量在 **48 小时** 内回应。

## 什么算漏洞？

- 可以绕过机器人权限执行任意命令
- 可以获取其他用户的聊天记录或好感度数据
- OTA 更新机制可以被劫持（中间人攻击）
- 可以导致机器人崩溃或资源耗尽
- 配置文件中敏感信息（API Key）泄露

## 什么不算漏洞？

- 机器人本身的 AI 回复不好笑
- 配置文件没填对导致启动失败
- 依赖包有已知漏洞但你没更新（请在 Issue 里提，不是安全问题）
- `/help` 命令信息不全
