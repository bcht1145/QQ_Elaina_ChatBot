# 命令列表
对于机器人的每一项命令均**集中**在此，或者，你也可以在QQ内输入`/help`查看  
管理员专用指令需要QQ号在配置文件中的`admin`列表内  
需要注意的是，每一项命令的前方均不需要at机器人  
OTA是每次启动时自动执行的  
### /help
用于获取命令列表

### /删除聊天记录
用于删除聊天记录，仅删除记忆，好感度和时间记录均不变

### /重置聊天记录
用于将用户的记录文件重置至默认值  
也就是相当于第一次聊的时候一样

### /撤回上一条
用于撤回你与机器人上一条聊天的记录

### /查看好感度
查看你与机器人之间的好感度，设定的好感度不会显示

### /查看上一条
列举出你与机器人上一条聊天的时间，聊了什么

### /AI 或 @机器人
用于触发AI聊天  
一般情况下，`@机器人`是用于群聊  
`/AI`是用于私聊
